Sobre o projeto:


Esse projeto se trata de um site de recomendação de peças para computadores feita para a matéria de Inteligencia Artificial do curso de Ciências da Computação do IFSC Campus Lages. Ele foi desenvolvido pelos alunos Alessandra Lima de Oliveira e Erik P. S. Borela sob o professor da disciplina Wilson Castello Branco Neto

O projeto foi desenvolvido usando redes Bayesianas. A rede usada para esse projeto é visível abaixo. Nela temos duas tabelas de input (Usage e Price) que ditam o proposito de uso e o quanto o usuário irá querer gastar com o computador. Então para cada componente (CPU, GPU e RAM), temos duas tabelas: uma para indicar o uso do componente e o componente escolhido de fato.
A partir dai a rede consegue nos dar o componente mais apropriado baseado nos dados que o usuário deu.


A rede foi inicialmente desenvolvida usando o software Netica e depois a rede foi convertida para ser usado com a biblioteca pgmpy.

O servidor do site então foi desenvolvido usando Flask, uma biblioteca para Python para a criação de servidores. Para o desenvolvimento do site em site foi usado a biblioteca para CSS materialize.css.

É feito então uma integração com o modelo do pgmpy com uma rota no Flask, que, ao enviar as informações de uso e preço a rota retorna os dados de uso e do componente escolhido. Com base nisso é então exibido as informações para o usuário junto com uma imagem e link para a compra do componente.

Todo o código fonte do projeto está hospedado no Github e pode ser acessado através desse link: https://github.com/erikborella/pc-recomendation-bayesian-network.
