Sobre o projeto:


Esse projeto se trata de um site de recomendação de peças para computadores feita para a matéria de Inteligencia Artificial do curso de Ciências da Computação do IFSC Campus Lages. Ele foi desenvolvido pelos alunos Alessandra Lima de Oliveira e Erik P. S. Borela sob o professor da disciplina Wilson Castello Branco Neto

O projeto foi desenvolvido usando redes Bayesianas. A rede usada para esse projeto é visível abaixo. Nela temos duas tabelas de input (Usage e Price) que ditam o proposito de uso e o quanto o usuário irá querer gastar com o computador. Então para cada componente (CPU, GPU e RAM), temos duas tabelas: uma para indicar o uso do componente e o componente escolhido de fato.
A partir dai a rede consegue nos dar o componente mais apropriado baseado nos dados que o usuário deu.
